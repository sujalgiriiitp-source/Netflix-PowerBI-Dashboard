# Netflix Power BI Dashboard

This project presents a Netflix Data Analysis Dashboard created using Microsoft Power BI.

## Project Overview
The dashboard analyzes Netflix content data and provides insights about:
- Top genres available on Netflix
- Ratings distribution of movies and TV shows
- Country-wise content distribution
- Yearly trend of Netflix titles

## Tools Used
- Microsoft Power BI
- Dataset (Netflix titles dataset)
- Data Visualization techniques

## Project Files
- Netflix Dashboardsujal.pbix – Power BI dashboard file
- netflixabhi.jpg – Dashboard screenshot

## Result
The dashboard provides interactive visualizations that help understand trends in Netflix content and enables users to explore the data easily.
